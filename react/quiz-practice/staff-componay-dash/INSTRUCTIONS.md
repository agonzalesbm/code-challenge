# The HackerEarth Mock Challenge: "The Employee Directory"

## Background

You are building a dashboard for a company to manage its staff. You need to fetch employee data, allow for real-time searching, and enable simple management actions.

## The Requirements

1. Fetch & Display: On mount, fetch a list of users from https://jsonplaceholder.typicode.com/users.
2. Search Functionality: Implement a search bar that filters users by name or email.
3. Delete Action: Each user card must have a "Delete" button that removes them from the local list.
4. Detail Toggle: Clicking on a user's name should "expand" the card to show their City and Company Name.
5. Empty State: If no users match the search, display a "No results found" message.
